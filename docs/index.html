About Bookshelf Manager


Bookshelf Manager is a vertical social network, dedicated to book lovers and all readers. Bookshelf Manager is the richest source of information on books and the most independent. Sign up and search Bookshelf Manager to find useful, interesting information on millions of books, share opinions with other readers, and see what other think by reading their reviews and opinions.

The application was written in SPA technology using the vue.js framework. The visual page was created with vutefiy. Firebase is responsible for the backend page, with which users can create accounts and also share data on books using Firestore.
